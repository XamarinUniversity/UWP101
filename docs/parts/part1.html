Exercise 1: Create and run a UWP app

The goal of this lab is to write a simple but functional UWP app: you will create a new project, add a minimal UI, implement a small piece of behavior, and then run the app on one or more devices/emulators.

You will be coding a Tip Calculator app. The app will have a single page containing a TextBox and a TextBlock (see below). The user will type the amount of their restaurant bill into the TextBox. Your code will react to the TextBox's TextChanged event to calculate a 15% tip and load the amount of the tip into the TextBlock.

The finished UI will look like this:

To complete this exercise, you will need a Windows development environment with Visual Studio. We recommend Windows 10 Anniversary Edition or higher because the exercise uses APIs that were added in that release. It is also possible to develop on a previous version of Windows and test on an emulator rather than the local device, please see this setup page for an example of how to do this. Note: You must enable Developer Mode on your development machine to work with UWP apps. Visual Studio will prompt you to do this the first time it is needed. You can also use these instructions if you would like to enable it ahead of time.

Create the app

  1. Open Visual Studio.
  2. Create a new UWP application using the Blank App template.
  3. Name the project Tip Calculator.

Create the UI

  1. Open MainPage.xaml.
  2. Change the root layout panel from a Grid to a StackPanel.
  3. Add a TextBox and a TextBlock.
  4. Add x:Name="billInput" to the TextBox element so you can access it from code behind.
  5. Add x:Name="tipOutput" to the TextBlock element so you can access it from code behind. 
<StackPanel Background="{ThemeResource ApplicationPageBackgroundThemeBrush}" HorizontalAlignment="Center" VerticalAlignment="Center">
   <TextBox x:Name="billInput" />
   <TextBlock x:Name="tipOutput" />
</StackPanel>

Implement behavior

  1. Subscribe to the TextBox's TextChanged event. You can do this in XAML (see below) or in code using the x:Name of the TextBox.
<TextBox x:Name="billInput" TextChanged="OnBillChanged" />
  1. Open MainPage.xaml.cs.
  2. In your TextChanged handler, retrieve the value from the TextBox, use it to compute a 15% tip, and load the tip into the TextBlock. 
void OnBillChanged(object sender, TextChangedEventArgs e)
{
  double bill;
  if (!double.TryParse(billInput.Text, out bill))
    return;
  double tip = bill * 0.15;
  tipOutput.Text = string.Format("{0:C}", tip);
}

Run the app

  1. If your development machine is running Windows 10, run the app on the local machine.
  2. If you have time, run the app on one of the emulators (e.g. a Mobile Emulator).

Congratulations!

This lab walked you through building a simple UWP app: you created a project, defined the UI in XAML, and wrote the code-behind. You then tested the app on the local machine and/or an emulator, which let you verify that a single UWP app can run on multiple Windows device types.
